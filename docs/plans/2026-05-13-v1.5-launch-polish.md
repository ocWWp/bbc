> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Tech Stack:** Next.js 16, React 19, Supabase (Postgres + RLS + Auth), Cloudflare Workers (via OpenNext), Anthropic Claude SDK, BlockNote editor, Tailwind. Security tools (off-the-shelf): Cloudflare WAF, Semgrep CI, Socket, Dependabot, OpenSSF Scorecard.

In `flag-action.ts`:

```ts
"use server";

import { requireActor } from "@/lib/auth/require-user";
import { getStore } from "@/lib/store";

export async function flagMemory(formData: FormData): Promise<{ ok: boolean; error?: string }> {
  const a = await requireActor();
  if (!a.ok) return { ok: false, error: a.output };

  const memoryId = String(formData.get("memory_id") ?? "");
  const reason = String(formData.get("reason") ?? "");
  if (!memoryId || !reason) return { ok: false, error: "memory_id and reason required" };

  const store = await getStore();
  // File a proposal — body cites the source memory and includes the flag reason.
  // The proposal targets the memory's owning_layer; admin reviews it in /queue.
  await store.queue.fileProposal({
    target_file: `memory/<id>/${memoryId}`,
    change_kind: "edit",
    summary: `Flag: ${reason.slice(0, 80)}`,
    body: `Flagged by ${a.actor.actor}:\n\n${reason}\n\n— filed from /brain/${memoryId}`,
    actor: a.actor.actor,
    tenant_id: a.actor.tenant_id,
  });
  return { ok: true };
}
```

(`store.queue.fileProposal` may not exist with that signature — check `@bbc/store` package; adjust to whatever the actual file-or-DB-mode signature is.)

In `ReadOnlyMemory.tsx`, add a small "Flag this" button that opens a modal with a textarea, submits to the action.

export async function readRecentDrafts(tenantId: string, role: string, limit = 10) {
  const supabase = await getSupabaseServerClient();
  const { data } = await supabase
    .from("studio_runs")
    .select("id, template_id, status, output, created_at")
    .eq("tenant_id", tenantId)

    .order("created_at", { ascending: false })

  return (data ?? []).map((r) => ({
